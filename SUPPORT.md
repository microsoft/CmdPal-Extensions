# Support

## How to file issues and get help

This project uses [GitHub Issues](https://github.com/microsoft/CmdPal-Extensions/issues) to track
bugs and feature requests. Please search the existing issues before filing new issues to avoid
duplicates. For new issues, file your bug or feature request as a new Issue.

For help and questions about building Command Palette extensions, see the
[Extension Development docs](https://learn.microsoft.com/en-us/windows/powertoys/command-palette/extensibility-overview).

For help submitting your extension to the gallery, see the
[Contributing Guide](docs/CONTRIBUTING.md).

## Microsoft Support Policy

Support for this project is limited to the resources listed above.
